VSE_DEBUG - сборка отладочной версии движка VSE, отключается останов движка при исключениях в пользовательском коде
VSE_LOG - включить лог
VSE_CONSOLE - включить консоль
VSE_NO_INI - не использовать ini-файл для настроек
PM_DRAWNORMALS - рисовать нормали у моделей
VSE_ESC_EXIT - принудительный выход по escape
VSE_USE_SNAPSHOT_KEY - сохранение скриншота по нажатию кнопки Print Screen
VSE_USE_ALT_ENTER - переключение фуллскрина по Alt-Enter
VSE_USE_MEMPAK_PAS - помещать данные мемпака в секцию переменных
